¿ Qué es Luwica ?

Es un dispositivo con arquitectura digital y funcionamiento sencillo; una lámpara que puede ser encendida y apagada por medio de una interfaz de usuario que se accede por medio de wifi y que se manipula en el celular.

Funcionamiento

El producto es una lámpara. La diferencia es que esta puede conectarse a internet y ser manipulada remotamente desde la red por medio de una aplicación en el celular. Cuando se enciende, esta debe ser configurada conectándola a la red del hogar como se suele hacer comúnmente cuando se desea conectar a internet algún dispositivo. Después de que la lámpara está conectada, se ingresa al aplicativo del dispositivo, esta consta de dos botones, normalmente para encender y apagarla.

Producción

Para el proyecto específicamente se consta de un relevador, un suiche y el ESP 8266. Todo este circuito se adapta a una lámpara para manipularla remotamente mediante la web.
